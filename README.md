Set up locally windows pc

-Frontend
 1) cd ./frontend
 2) npm i
 3) npm run serve

-Backend
 1) cd ./backend
 2) pip install -r requirements.txt
 3) python -m venv venv
 4) venv\Scripts\activate #on windows
 5) python app.py
